# Action Plugin

[p] is your prefix.
add it by doing [p]plugins add 6days9weeks/modmail-plugins/action

# commands
**Eg:** [p]hug user
baka - Call a user baka!
cuddle - Cuddle a user!
feed - Feed a user!
hug - Hug a user!
kiss - Kiss a user!
pat - Pat a user!
poke - Poke a user!
slap - Slap a user!
smug - Smug at a user!
tickle - Tickles a user!
